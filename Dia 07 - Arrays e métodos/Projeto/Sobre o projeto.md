Arquitetura Proposta e Fluxo Lógico (Sem Código-Fonte)
​Contrato de Dados (Input): Crie uma constante global com um array estático de 6 objetos. Cada objeto representará um FII com as propriedades: ticker (string), sector (string), price (number) e dividendYield (number).
​Camada de Filtro Lógico: Crie uma função pura chamada generateLogisticsReport(portfolio). Internamente, essa função deve aplicar .filter() na coleção para reter exclusivamente os objetos cujo sector seja exatamente igual a 'Logística'.
​Camada de Transformação: No mesmo fluxo de dados, encadeie o .map() no resultado do filtro. A função de transformação deve converter cada objeto filtrado em uma string estruturada (ex: template literals retornando "O ativo TICKER custa R$ PRECO e pagou YIELD%").
​Camada de Saída (Output): Receba o array de strings final. Percorra esse array (aqui o uso de .forEach() ou console.log() direto do array é aceitável) para imprimir cada linha do relatório no terminal.
